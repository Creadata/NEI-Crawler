NEI-Crawler Tool - User Guide

Introduction

The NEI-Crawler Tool is designed to streamline the extraction and classification of Named Entities of Interest (NEIs) from large datasets, particularly in the context of cybercrime data analysis. It leverages Named Entity Recognition (NER) techniques and custom Regular Expressions (RegEx) to identify and categorize entities relevant to Digital Forensics and Cyber Investigation (DFCI).

Features

  • Data Upload: Upload raw data files such as text logs, chat transcripts, or other unstructured data sources.
  • NEI Configuration: Define custom NEI groups and associate them with specific RegEx patterns.
  • Entity Recognition: Scan uploaded data to extract and classify entities based on NEI groups.
  • Results Visualization: View extracted entities grouped by NEI categories.
  • Export Options: Export the results in various formats compatible with other forensic tools.
  • Integration Capabilities: Interfaces to integrate with existing digital forensic tools.
  • Privacy Compliance: Runs locally to ensure data privacy and compliance with regulations.

Getting Started

1. Data Upload

  • Navigate to the Data Upload section.
  • Click the "Choose File" button to select your data file (supported formats: .txt, .csv, .json).
  • Click "Upload" to submit the file to the tool.

2. NEI Configuration

  • Go to the NEI Configuration section.
  • Add NEI Groups:
    • Click "Add NEI Group".
    • Enter a unique name for the NEI group (e.g., "IP Addresses").
  • Assign RegEx Patterns:
    • For each NEI group, input the RegEx patterns that match the entities of interest.
    • Save the configurations.

3. Entity Recognition

  • Navigate to the Entity Recognition section.
  • Click "Start NER Process" to begin scanning the uploaded data.
  • The tool will process the data and extract entities based on your NEI configurations.
  • A status message will indicate when the process is complete.

4. Viewing Results

  • Go to the Results section.
  • View the extracted entities grouped by NEI categories.
  • Use the search or filter options to locate specific entities.

5. Exporting Results

  • In the Results section, click the "Export Results" button.
  • Choose the desired format for export (e.g., .csv, .json).
  • The results will be downloaded to your local machine.

6. Settings and Integration

  • Access the Settings section to configure integrations with other forensic tools.
  • Set preferences for data formats and interface options.

Tips

  • Valid RegEx Patterns: Ensure your RegEx patterns are correct to avoid missing entities.
  • Data Privacy: Since the tool runs locally, your data remains secure on your machine.
  • Regular Updates: Keep your NEI configurations updated based on evolving investigative needs.
